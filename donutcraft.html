Getting Started

Main Idea

I do not want to hold players hand through out the pack; I want players to be able

to build what they want and all that, but I have many recipe changes to make things

a little more challegneing. One of them being how machines are made. So bascially you

are going to need to get started in IC2, Forestry and TechReborn, from there you can go to Thermal Expansion.

Now this may take some time, but once you got somewhat far in each mod and have enough

resources you can start working on an AE2 system. Then you will be able to go in and

start automating everything with AE2.

Spawning in

Just like any other pack you are going to want to go around and find materials.

I haven't really altered much with early game stuff and not sure if I am going to

because as of right now it seems to be balaning out with the challenges later.

First Machines

You need to start by making the Iron Alloy Furnace, from there you make bronze so you

can get the carpenter and you will also want to make a clock work engine to power the

carpenter. After that you can start making circuits so you can start making machines like

the electric alloy smelter, macerator, batbox, etc.

The reset is up to you, play around and test things to keep progessing throughout the pack and mods.
